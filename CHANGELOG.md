# Changelog

## v0.1.0 — Tactical intake shell

- Added dependency-free browser application.
- Added tactical room grid and 2.5D object renderer.
- Added OBJ and STL upload support.
- Added primitive cube and marker spawning.
- Added selection, move, rotate, scale, duplicate, hide/show, and despawn controls.
- Added Minecraft-style slash command console.
- Added ARC-Room scene JSON import/export.
- Added object receipts for command history.
- Added optimized screenshot assets for GitHub README display.
